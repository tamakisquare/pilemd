# PileMd

https://pilemd.com/

![PileMd](https://pilemd.com/images/top1.png)

## Run

```
npm install
NODE_ENV=development webpack
Electron .
```

## Build

* `gulp`
* `PM_OSX_SIGN='...' gulp electron`
* `gulp electron-linux`
* `gulp electron-windows`
